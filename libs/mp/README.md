# csl::mp

C++ **TMP** (template metaprogramming) utilities.

Part of [CppShelf](https://github.com/GuillaumeDua/CppShelf) — a collection of single-header, header-only C++ libraries.

## Include

```cpp
#include <mp/csl/mp.hpp>
```

## Overview

`csl::mp` provides a collection of metaprogramming primitives and algorithms operating on type lists, including:

- Type sequence manipulation (`mp::type_list`, `mp::flatten`, `mp::filter`, ...)
- Type predicates and transformations
- Compile-time tuple utilities

Documentation is a work in progress.
See also [`doc/WIP.md`](doc/WIP.md) for in-progress notes.
