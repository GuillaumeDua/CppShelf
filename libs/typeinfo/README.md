# csl::typeinfo

Type introspection utilities for C++23.

Part of [CppShelf](https://github.com/GuillaumeDua/CppShelf) — a collection of single-header, header-only C++ libraries.

## Include

```cpp
#include <typeinfo/csl/typeinfo.hpp>
```

## Overview

`csl::typeinfo` provides compile-time and runtime type introspection helpers, including human-readable type names without RTTI overhead.
