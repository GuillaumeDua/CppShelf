# csl::ensure

Assertion and contract utilities for C++17/20/23.

Part of [CppShelf](https://github.com/GuillaumeDua/CppShelf) — a collection of single-header, header-only C++ libraries.

## Include

```cpp
#include <ensure/csl/ensure.hpp>
```

## Overview

`csl::ensure` provides lightweight precondition and postcondition checks with configurable violation handling.
